# Mental Health and Burnout in Tech Workers 2026

Анализ факторов выгорания и увольнения сотрудников в IT.

Датасет:
https://www.kaggle.com/datasets/mohankrishnathalla/mental-health-and-burnout-in-tech-workers-2026

Что сделано:
- EDA
- Обработка выбросов
- Feature engineering
- Обучение моделей
- Оценка качества моделей

Модели:
- Logistic Regression
- Random Forest

Метрики:
- Accuracy
- ROC-AUC
- F1-score

Основные библиотеки:
- pandas
- numpy
- matplotlib
- scikit-learn

Запуск:

pip install -r requirements.txt

jupyter notebook
